（１）モデルパラメータを既知として与えた場合
MAIN1.mがメインプログラムである．
FunUsf.mはその中で引用される都市貯留関数モデルのプログラムである．
rq36.datが入力ファイルで，1分間隔の流域平均雨量と観測流出高の2列で構成されるデータファイルである．
上記３つのファイルを同じフォルダーに格納してMAIN1.mを実行する．
実行すると，ファイル名usfrkg.outが作成され，その中に計算流出高が，入力データの流域平均雨量，観測流出高とともに出力される．
なお，参考のため本プログラムを実行した時の出力結果をファイル名MAIN1.outに格納している．

（２）モデルパラメータを同定して洪水流出解析を行う場合
MAIN2.mがメインプログラムである．
FunUsfRkg.mはMAIN2.mの中で引用されるRunge-Kutta-Gill法による都市貯留関数プログラムで，RMSEを計算するプログラムである．FunUsfRkg.mの中で（１）で用いたFunUsf.mを引用している．
FunHydro.mはMAIN2.mの中で引用される最適パラメータによる結果を出力するプログラムである．
入力ファイルは（１）同様rq36.datである．
上記５つのファイルを同じフォルダーに格納してMAIN2.mを実行する．
実行すると，ファイル名hydro.outが作成され，その中に最適パラメータによる各時点の計算流出高が，入力データの流域平均雨量，観測流出高とともに出力される．
なお，参考のため本プログラムを実行した時の出力結果をファイル名MAIN2.outに格納している．
注意事項として，本プログラムでは結果の再現性を保証するため，MAIN2.mの60行目のプログラムで，乱数発生の初期値を固定しており，プログラムを実行するたびに毎回同じ乱数が発生するようにしている．別の乱数を発生させて実行するには，その行内の数値0を変更する必要がある．
